DLSITENameGetter
================

Adjust folder name from DLSITE.com

ex:

BEFORE:

C:\DL\

-- C:\DL\RJ012345abcd

-- C:\DL\RJ012346ssd

-- C:\DL\RJ012347asdf

-- C:\DL\rj.py

run python rj.py C:\DL

AFTER:

C:\DL\

-- C:\DL\[GROUP Name]TITLE RJ012345

-- C:\DL\[GROUP Name]TITLE RJ012346

-- C:\DL\[GROUP Name]TITLE RJ012347

-- C:\DL\rj.py
